#  ISO 27001 Gap Analysis Project – NovaPay Fintech

##  Overview
This project presents a comprehensive ISO/IEC 27001 Gap Analysis conducted for a simulated fintech organization, NovaPay. The objective was to assess the organization's current security posture, identify compliance gaps and recommend actionable remediation strategies.

---

##  Objectives
- Evaluate existing security controls against ISO 27001 requirements  
- Identify gaps in governance, access control, incident management and data protection  
- Provide structured recommendations for compliance improvement  
- Develop a remediation plan with ownership and timelines  

---


---

##  Key Findings
- Lack of Multi-Factor Authentication (MFA)  
- Absence of formal Incident Response process  
- Incomplete data encryption across systems  
- Weak access control implementation (No RBAC)  
- No Business Continuity Plan (BCP)  

---

##  Recommendations
- Implement MFA across critical systems  
- Develop and enforce an Incident Response Plan  
- Apply encryption to sensitive data  
- Introduce Role-Based Access Control (RBAC)  
- Establish Business Continuity and Disaster Recovery plans
